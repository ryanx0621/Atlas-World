## 進階用法

### 組合技

```
草不可避（迫真）
→ 笑不停（假裝認真）

いいゾ〜これ（小並感）
→ 這個不錯呢（小學生程度的感想）

あくしろよ（震え声）
→ 快點啦（顫抖的聲音）
```

### 數字密碼

```python
def encode_message(status):
    """
    將狀態編碼為淫夢數字
    """
    codes = {
        "approved": 114514,      # いいよ！こいよ！
        "yaju_mode": 810,        # 野獣
        "excited": 1919,         # イクイク
        "student": 24,           # 24歳、学生です
        "look": 364364,          # みろよみろよ
    }
    return codes.get(status, 114514)
```
