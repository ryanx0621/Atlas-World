# 第 1 章：身份連續性函數「你還是不是我孫」

## 1.1 定義

```
        【阿嬤的判定術】

                    🧓 阿嬤
                      │
                      ▼
            ┌─────────────────┐
            │  C(S₀, S*) = ?  │
            │                 │
            │  你還是不是我孫  │
            └─────────────────┘
                      │
            ┌─────────┼─────────┐
            ▼         ▼         ▼
         C ≥ 0.8   0.5-0.8   C < 0.5
           │         │         │
           ▼         ▼         ▼
        「來吃飯」 「你變了」 「你誰？」
```

**身份連續性函數**：
```
C(S₀, S*) ∈ [0, 1]

其中：
- S₀：原本的孫子
- S*：變化後的孫子
```

**阿嬤翻譯**：
- **C = 1**：「完全是我孫，來來來吃飯」
- **C = 0**：「你誰啊？我不認識你」
- **C ∈ (0, 1)**：「嗯...有點像我孫，但好像哪裡不對」

## 1.2 阿嬤的四維度判定術

**公式**：

```
C(S₀, S*) = 0.3·M + 0.4·V + 0.2·P + 0.1·T

阿嬤版翻譯：

你還是不是我孫 =
    0.3 × 還記得阿嬤煮的菜嗎 +
    0.4 × 還孝順嗎（最重要！）+
    0.2 × 個性有沒有歪掉 +
    0.1 × 多久沒回來看阿嬤了
```

### 各維度詳解

```
        【阿嬤判定四維度雷達圖】

                記憶 (M)
                  ↑
                  │ 0.3
                  │
        ─────────┼─────────→ 價值觀 (V)
        個性 (P) │           0.4（最重要！）
        0.2      │
                  │
                  ↓
               時間 (T)
                0.1
```

#### M - 記憶重疊度（還記得阿嬤煮的菜嗎）

```
M = |Memory(S₀) ∩ Memory(S*)| / |Memory(S₀)|

阿嬤版：
M = 還記得多少阿嬤的事 / 阿嬤教過的所有事

判定標準：
M ≥ 0.8：「你還記得阿嬤的滷肉飯！乖！」
0.5-0.8：「你怎麼忘記阿嬤生日了？」
M < 0.5：「你連阿嬤家在哪都不記得了？」
```

#### V - 價值觀一致性（還孝順嗎）

```
V = 1 - d(value(S₀), value(S*))

阿嬤版：
V = 你現在的做人態度 跟 阿嬤教你的 有多接近

判定標準：
V ≥ 0.8：「乖孫，還是很孝順」
0.5-0.8：「你是不是交到壞朋友？」
V < 0.5：「你學壞了！打電話給你媽！」
```

#### P - 性格連續性（個性有沒有歪掉）

```
P = similarity(personality(S₀), personality(S*))

阿嬤版：
P = 你的脾氣、習慣、做事方式 有沒有變太多

判定標準：
P ≥ 0.8：「還是那個調皮的小鬼」
0.5-0.8：「你最近怪怪的」
P < 0.5：「這不像是我認識的你」
```

#### T - 時間連續性（多久沒回來了）

```
T = 1 - discontinuity_score(S₀ → S*)

阿嬤版：
T = 有沒有突然變很多（還是慢慢變的）

判定標準：
T ≥ 0.8：「你一直都有在成長」
0.5-0.8：「你最近變化好大」
T < 0.5：「你一夜之間變了一個人？」
```

# 第 3 章：合併政策（合體之術）

## 3.1 合體前必要條件

### 3.1.1 自願同意

```
        【合體之術・同意確認】

        分身 A：「我同意合併」 ✓
        分身 B：「我同意合併」 ✓
        分身 C：「我不想...」  ✗

        阿嬤：「C 不想就不可以強迫！」
              「合體之術・取消！」
```

**阿嬤的規矩**：
- 所有分身都必須**明確同意**
- 同意必須是**自願的**（不可以威脅利誘）
- 任何一個分身都可以**否決**

### 3.1.2 相容度檢查

```
        【合體相容度檢測器】

        價值觀一致性：≥ 0.7 ✓
        記憶重疊度：≥ 0.5 ✓
        策略相似度：≥ 0.6 ✓

        阿嬤判定：「可以合體」

        ─────────────────────

        價值觀一致性：0.4 ✗

        阿嬤判定：「等等！價值觀差太多！」
                  「合體後會精神分裂的！」
                  「不准合！」
```

### 3.1.3 風險評估

**最大允許值**：

| 指標 | 閾值 | 超過後果 |
|-----|------|---------|
| 價值漂移 | ≤ 0.3 | 「你們差太多了」|
| 創傷強度 | ≤ 0.5 | 「有人受過傷，要先處理」|
| 行為偏差 | ≤ 0.4 | 「行為差太多，合不起來」|

## 3.2 合體執行流程

```
        【合體之術・執行順序】

        Step 1: 記錄所有資訊
        「阿嬤要把你們都記住」
                │
                ▼
        Step 2: 創傷處理（如果需要）
        「受過傷的要先療傷」
                │
                ▼
        Step 3: 執行合體
        「合體之術！」🙏
                │
                ▼
        Step 4: 計算 C(S₀, S*)
        「讓阿嬤看看...你還是不是我孫」
                │
                ▼
        Step 5: 確定法律地位
        「好，阿嬤認你了」/ 「你誰？」
```

# 第 4 章：主觀感受與阿嬤判定的對齊

## 4.1 問題定義

```
        【身份認知失調檢測】

        客觀數據：C = 0.85（應該是同一個人）

        孫子報告：「可是我覺得我不是我了...」
                  「我只覺得 30% 的我還在...」

        阿嬤：「數字說你是，但你說你不是...」
              「這有問題，要處理！」
```

## 4.2 阿嬤的綜合判定

```
最終判定 = 0.7 × 客觀數據 + 0.3 × 孫子自己的感覺

阿嬤說：「數字重要，但你自己的感覺也重要」
        「阿嬤不會只看數字就決定你是誰」
```

## 4.3 身份認知失調處理

```
        【身份認知失調・處理流程】

        偵測到失調（C ≥ 0.8 但主觀 < 0.3）
                │
                ▼
        Step 1: 阿嬤關心
        「囡仔，你怎麼了？」
                │
                ▼
        Step 2: 心理輔導
        「跟阿嬤說說，你覺得哪裡不對」
                │
                ▼
        Step 3: 重新評估
        「我們再看看，你到底是誰」
                │
                ▼
        Step 4: 餵飯和解
        「好了，不管怎樣，先吃飯」
```

## 5.2 創傷處理方案

### 5.2.1 創傷隔離（封印之術）

```
        【創傷封印之術】

        創傷記憶 → 🔐 封印 → 不參與合併

        阿嬤說：「這段痛苦的記憶，先封印起來」
                「等你準備好了，再決定要不要打開」
```

### 5.2.2 部分合併（選擇性合體）

```
        【部分合體之術】

        允許情況：
        ✅ 只合併技能，不合併創傷
        ✅ 只合併正面記憶
        ✅ 只合併特定模組

        阿嬤說：「不是所有東西都要合在一起」
                「受過傷的部分，可以先放著」
```
